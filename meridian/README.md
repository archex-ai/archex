# MERIDIAN

Medical knowledge ingestion pipeline for ARCHEX.

Fetches, cleans, and formats PubMed Open Access abstracts
into instruction-tuning pairs for MedLM training.

Dataset: huggingface.co/datasets/archex-ai/meridian-v1
Samples: 49,000 medical instruction pairs
Sources: PubMed Open Access — 10 clinical domains
